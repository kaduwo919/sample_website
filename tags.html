重要なHTMLタグ一覧

このページは、初心者シリーズWeb基礎編「 HTML」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページとして作られます。このページの制作を通じて、HTMLのテーブル（table）をtableタグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ブロック要素（block element）

HTMLタグ タグ名 目的
h1–h6 見出し（heading） この中に見出しを置く（レベル：1–6）
p 段落（パラグラフ：paragraph） この中に文章を入れる
table 表（テーブル：table） この中にテーブルを置く
tr 表の行（table row） この中に行データを置く
th 表の見出し（table header） この中に表の見出しを置く
td 表のデータ（table data） この中に表のセルデータを置く
header ヘッダー（header） ヘッダーを表す
div 段落分け（divided） ブロックレベル要素としてのグループ化
ol 順序のあるリスト（ordered list） 順序のある番号付きのリストを表示
ul 順序のないリスト（unordered list） 順序のない箇条書きのリストを表示

インライン要素（inline element）

HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調（emphasized） テキストを軽く強調する 技術の成熟（<em>technical sophistication</em>） 技術の成熟（technical sophistication）
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー（anchor） ハイパーリンクにする <a href="https://learnenough.com/>Learn Enough</a> Learn Enough
img 画像（image） 画像を配置する <img src="https://bit.ly/1MZAFuQ" alt="Michael HEartl">
code ソースコード（code） プログラムコードを示す <code>code</code> code
span スパン（span） インライン要素としてグループ化 <span>hello</span>,world! hello,world
li リスト項目（list item） リストの項目を表示 <li>1</li><li>2</li>
